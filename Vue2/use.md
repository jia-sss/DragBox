```html
<drag-box>
    <drag-box-item>
        <div> DIV1 </div> 
    </drag-box-item>
    <drag-box-item>
        <div> DIV2 </div> 
    </drag-box-item>
    <drag-box-item :resizeShow="false">

    </drag-box-item>
```

## drag-box-item Attributes
<br/>

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| ---- | ---- | ---- | ---- | ----- |
| resizeShow | 是否显示拖动条 | boolean | - | true |
